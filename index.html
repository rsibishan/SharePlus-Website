React Native

SHARE+ is developed by React and React-Native frameworks, which was introduced by Facebook.

Native Base

User Interface(UI) is designed using Native Base, which made our app, SHARE+, more prettier.

Firebase

Database of SHARE+ is fully handled by Firebase, which is provided by Google.

Expo.io

SHARE+ is fully developed with Expo platform tools and published by Expo.io
